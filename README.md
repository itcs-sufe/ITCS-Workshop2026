# ITCS Workshop 10th Anniversary Website

Welcome to the official website repository for the **ITCS Workshop 10th Anniversary Celebration**. This project is a responsive, single-page website designed to showcase event details, schedule, speakers, and organization information for the workshop.

## 🚀 Features

-   **Responsive Design**: Optimized for both desktop and mobile viewing with a clean, modern interface.
-   **Interactive Navigation**: Smooth scrolling to different sections (Home, Background, Schedule, Speakers, Organization).
-   **Image Slider**: A JavaScript-powered auto-scrolling image carousel for showcasing event highlights.
-   **MathJax Support**: Integrated MathJax for rendering mathematical notations beautifully.
-   **Modern Typography**: Utilizes Google Fonts (Cormorant Garamond, Lato) for elegant typography.

## 🛠️ Technology Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling, Flexbox for layout, and animations.
-   **JavaScript**: Dynamic behavior for the navigation menu (mobile toggle) and image slider.

## 📂 Project Structure

```
ITCS-10th-Anniversary/
├── index.html    # Main entry point
├── style.css     # Stylesheets
├── script.js     # Logic for slider and mobile menu
└── README.md     # Project documentation
```

## 📦 Usage

To view the website locally:

1.  Clone the repository.
2.  Open `index.html` in your preferred web browser.

## 🌐 Deployment

This website is ready to be hosted on **GitHub Pages**.

1.  Push the code to a GitHub repository.
2.  Go to repository **Settings** -> **Pages**.
3.  Select the `main` branch as the source and save.
4.  Your site will be live at `https://<username>.github.io/<repo-name>/`.
